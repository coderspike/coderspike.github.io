一 、常用结构体

  • NSRange:范围结构体，用来对对象进行范围操作。
  • NSPoint：表示一个点。
  • NSSize :表示宽高
    struct NSSize {
    CGFloat width;
    CGFloat height;
    };
  • CGRect: 用来表示一个矩形的位置和宽高

struct CGRect {
CGPoint origin;

};
